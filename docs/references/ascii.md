# ASCII diagram principles

instructions for agent to make ascii diagrams.

Create a **beautiful, clean, and professional** ASCII diagram using these strict style rules:

**Style Requirements:**
- Use thick box drawing characters: ╔ ═ ╗ ║ ╚ ╝ ╠ ╣ ╬ ╦ ╩
- Prefer clean, modern look with good spacing and visual hierarchy
- Center main titles
- Keep the entire diagram under 110 characters wide
- Use consistent alignment
- Add a small legend at the bottom when needed
- Use proper arrows: ──▶  ◀──  ↑  ↓  ↔
- Add subtle separators when grouping sections

Make it clear enough for a junior developer and elegant enough for a tech lead.
